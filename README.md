# Temperature Check

A small interactive web application where the user enters the outdoor temperature and gets a message based on simple conditions.

---

## Features

- Accepts a temperature value as user input
- Displays a message based on temperature ranges
- Uses a small function to map temperature → message
- Re-renders the view using a simple MVC-inspired structure

---

## Focus

- if / else if / else
- Comparison operators and ranges
- Converting input values to `Number`
- Global state variables + re-rendering (`updateView()`)

---

## Run

Open `index.html` with Live Server.
